# Smart-Urban-Delivery-Optimization-System
Overview

SUDOS is a Design and Analysis of Algorithms (DAA) project that models urban delivery systems using graphs.
It optimizes delivery routes, agent assignment, and cost/time using multiple algorithmic approaches.

---

Objectives

- Apply Greedy, Dynamic Programming, and Graph algorithms
- Model real-world delivery problems using graphs
- Compare optimal and approximate solutions
- Analyze time and space complexity

---

Problem Modeling

- Graph-based system
  - Nodes → Locations
  - Edges → Distance/Time
- Routes assigned to delivery agents
